# FF14 职业输出轴（拖拽版）

这个版本专注于“按职业排输出轴”：

- 先选择职业（默认 SMN）
- 内置职业技能图标（玩家无需手动上传）
- 可继续维护技能库（技能名 / GCD / 类型）
- 技能区分为 `GCD 技能` 和 `能力技能`
- 按副本机制时间轴，拖拽技能到对应机制点
- 支持导入副本 `XLSX` 时间轴（读取首个工作表）
- 严格规则：
  - GCD 技能严格遵循 GCD 间隔
  - 能力技能必须在 GCD 技能之后
  - 每两个 GCD 技能之间最多插入 2 个能力技能
- 可导入/导出整套方案（职业、技能、机制点、输出轴）

## XLSX 时间轴格式

首个工作表中，每行一条机制，支持两类时间格式：

- `分:秒:厘秒`（例如 `1:42:00`）
- Excel 小数日期（例如 `0.024490741`，会自动换算秒）

并自动识别阶段标记（`P1/P2/P3...`）拼接到机制名。

建议至少包含：

- 时间列：`time` / `时间` / `sec` / `seconds`
- 名称列：`name` / `机制` / `event` / `标题`

例如：

| time | name |
| --- | --- |
| 15.0 | 开场读条 |
| 32.5 | 分摊点名 |

> 默认已集成示例职业技能图标与副本时间轴，玩家打开网页即可直接拖拽排轴。

## 本地运行

```bash
python3 -m http.server 8000
```

访问 `http://localhost:8000`。
