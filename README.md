# FF14 职业输出轴（拖拽版）

这个版本专注于“按职业排输出轴”：

- 先选择职业（默认 SMN）
- 维护技能库（技能名 / GCD / 图标上传）
- 技能区分为 `GCD 技能` 和 `能力技能`
- 按副本机制时间轴，拖拽技能到对应机制点
- 严格规则：
  - GCD 技能严格遵循 GCD 间隔
  - 能力技能必须在 GCD 技能之后
  - 每两个 GCD 技能之间最多插入 2 个能力技能
- 可导入/导出整套方案（职业、技能、机制点、输出轴）

> 目前技能图标与技能 GCD 数据由你手动维护，后续可直接替换成你上传的游戏实装数据。

## 本地运行

```bash
python3 -m http.server 8000
```

访问 `http://localhost:8000`。
